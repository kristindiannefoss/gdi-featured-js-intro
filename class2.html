Variable Scope

Variables have different scopes. Scope can also be called binding.

Global Scope

A variable declared outside a function has a global scope and can be accessed anywhere, even in a function.


let awesomeLanguage = 'JavaScript'; // global scope

function whatIsAwesome() {
	// the inner scope has access to outer scope
	console.log(awesomeLanguage + ' is pretty awesome.');
}

whatIsAwesome(); // 'JavaScript is pretty awesome.'

Local Scope

A variable declared within a function has a local scope and can only be accessed within that function.


function whatIsAwesome() {
	let awesomeLanguage = 'JavaScript'; // local scope
	console.log (awesomeLanguage + ' is pretty awesome.');
}

whatIsAwesome(); // 'JavaScript is pretty awesome.'

console.log(awesomeLanguage + ' is pretty awesome.');
// Error: undefined local variable or method 'awesomeLanguage'

Block Scope

let: limited in scope to the block, statement, or expression.

let keyword: defines a variable globally/locally to an entire function regardless of block scope.

```js function varTest() { var x = 1; if (true) { var x = 2; // same variable! console.log(x); // 2 } console.log(x); // 2 } function letTest() { let x = 1; if (true) { let x = 2; // different variable console.log(x); // 2 } console.log(x); // 1 } ```

Comparison Operators

Example Name Result
a == b Equal TRUE if a is equal to b (can be different types).
a === b Identical TRUE if a is equal to b, and the same type.
a != b Not equal TRUE if a is not equal to b (can be different types).
a !== b Not identical TRUE if a is not equal to b, or they are not the same type.
a < b Less than TRUE if a is strictly less than b.
a > b Greater than TRUE if a is strictly greater than b.
a <= b Less than or equal to TRUE if a is less than or equal to b.
a >= b Greater than or equal to TRUE if a is greater than or equal to b.

The if/else statement

Use else to provide an alternate set of instructions.


let age = 28;
if (age >= 16) {
	console.log ('Yay, you can drive!');
} else {
	console.log ('Sorry, but you have ' + (16 - age) +
	' years until you can drive.');
}

The if/else if/else statement

If you have multiple conditions, you can use else if.


let age = 20;
if (age >= 35) {
	console.log('You can vote AND hold any place in government!');
} else if (age >= 25) {
	console.log('You can vote AND run for the Senate!');
} else if (age >= 18) {
	console.log('You can vote!');
} else {
	console.log('You can\'t vote, but you can still write your representatives.');
}

While loops

While will repeat the same code over and over until some condition is met.


let bottlesOfBeer = 99;
while (bottlesOfBeer >= 1) {
	console.log (bottlesOfBeer + ' bottles of beer on the wall');
	bottlesOfBeer = bottlesOfBeer - 9;
}

For loops

For loops are very similar, but you declare a counter in the statement.


//will count 1 to 10
for (let i = 1; i <= 10; i++) {
	console.log (i);
}

Loops and logic

You can add other statements or logical operators inside the loops.


//Count from 1 to 50

for (let i = 1; i <= 50; i++) {
	console.log (i);
	if (i % 3 == 0) {
		// logs 'Buzz' if i is a multiple of three
		console.log (' Buzz');
	}
	if (i % 5 == 0) {
		// logs 'Bang' if i is a multiple of five
		console.log (' Bang');
	}
}

Break

To exit a loop, use the break statement.


//Count from 100 to 200

for (let i = 100; i <= 200; i++) {
	console.log('Testing ' + i);
	if (i % 7 == 0) {
		console.log('Found it! ' + i);
		break; // break out of the loop at the first multiple of 7
	}
}

Arrays

Arrays are ordered lists of values.


let arrayName = [element0, element1, ...];

You can put different types of data into an array.


let colors = ['Red', 'Orange', 'Yellow', 'Green'];
let lotteryNumbers = [33, 72, 64, 18, 17, 85];
let myFavoriteThings = ['Broccoli', 1024, 'Sherlock'];

Array Length

The length property tells you how many things are in an array


let colors = ['Red', 'Orange', 'Yellow', 'Green'];
console.log(rainbowColors.length);

Using Arrays

You can access items with "bracket notation" by using the position of the object you want. The first position is 0.


let colors = ['Red', 'Orange', 'Yellow', 'Green'];
let firstColor = rainbowColors[0]; // 'Red';
let lastColor = rainbowColors[3]; // 'Green';
let noColor = rainbowColors[4]; // undefined

Changing arrays

You can use bracket notation to change an item in an array


let myFavoriteThings = ['Plants', 2048, 'Star Wars'];
myFavoriteThings[2] = 'Flowers';
console.log(myFavoriteThings); // What is logged to the console?

Expanding arrays

Arrays do not have a fixed length. You can use "push" to add something to an array.


let myFavoriteThings = ['Plants', 2048, 'Star Wars'];
myFavoriteThings.push('Flowers');
console.log(myFavoriteThings); // What is logged to the console?
