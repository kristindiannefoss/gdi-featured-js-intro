Introductions!

What is JavaScript?

  • Written in 10 days by Brendan Eich in 1995
  • Executes in browsers by JavaScript engines
    • V8, SpiderMonkey, Nitro, Chakra
  • Standardized by EcmaScript specifications
    • JavaScript != EcmaScript
  • Executes server side using the Node.js runtime
  • JavaScript works alongside CSS and HTML

Let's Develop It

Let's write your first JavaScript program. Make a folder called gdi. Inside, make a new page called index.html. Place this code inside.


	<!DOCTYPE html>
	<html>
	<head>
	<title>Test Page</title>
	</head>
	<body>
	<p>This is my awesome JavaScript code.</p>
	<script>
	alert('Hello World!');
	</script>
	</body>
	</html>

Script Tags

You can mix JavaScript and HTML. The script tag tells your browser the stuff inside is code, not content.


	<script>
	CODE GOES HERE
	</script>

JavaScript Files

Just like CSS, you can split a long block of JavaScript into its own file. 


	<script src="path/to/file.js"></script>

Separating Instructions

After each individual statement, you can add a semicolon.


	<script>
	console.log('Hello World!');
	console.log('I am glad to meet you')
	console.log('I am fuzzy');
	</script>

Comments

You can leave comments in your code—notes that people can read and computers will ignore.


	<script>
	/*I can wrap long comments
	with multiple lines
	like this*/
	console.log('Hello World!'); //Or mark short comments like this
	</script>

Getting results onto your screen

Open a popup box.


					alert('Hello World!');

Display a message in your console.


					console.log('Hello World!');

Add something to the page.


					document.write('Hello World!');

Variables

A variable is a place to store values

  • var, let, const
  • are undefined unless assigned a value
  • holds information
  • can change over time

Using variables


	<script>
	let oranges = 5;
	let apples; // undefined

	apples = 2;
	console.log(`I need to buy ${apples + oranges} pieces of fruit.`);
	</script>

Let's Develop It

In your JS file, create a variable and give it a valid name and a value.

  • Display it in the console
  • Display it with an alert
  • Display it on the page (in different styles?)
  • Display it when a button is clicked
  • Display it, and hide it when a button is clicked
  • Change the color of it when the mouse hovers over it

Numbers

Variable can be numbers, either integers or floats (decimals).


	<script>
	let oranges = 5;
	let acidityLevel = 9.7;
	oranges * aciditylevel = 48.5
	</script>

The browser will automatically convert integers to floats if needed

Arithmetic Operators


	<script>
	var oranges = 5;
	var apples = 4;

	oranges + apples; // 9
	oranges - apples; // 1
	oranges * apples; // 20
	oranges / apples; // 1.25
	oranges % apples; // 1
	</script>

Let's Develop It

Create two variables and try some arithmetic operators. Don't forget to display your results!

Strings

Variable can be strings, groups of characters. You put your string in quotes.

ES6 introduced template strings which allow us to do string interpolation.


	<script>
	let city = 'Denver';
	let state = "Colorado";
	let statement = `${city} is the capital of ${state}`;
	console.log('I\'d like to use an apostrophe and hate double quotes');
	</script>

String Operators

You can put strings together with a +, the concatenation operator.


	<script>
	let city = 'Denver';
	let fullName = kittensName + ', CO';
	console.log(fullName); // 'Denver, CO'
	</script>

String Operators

Change case using `toUpperCase()` or `toLowerCase()`


	<script>
	let city = 'Denver';
	city.toUpperCase(); // DENVER
	city.toLowerCase(); // denver
	</script>

Functions

Functions are separable, reusable pieces of code.

They hold a set of instructions.

Using Functions

First, declare the function.


	<script>
		function squidFact() {
			console.log("Squid are ancient animals.");
		}
	</script>

Then, use it as many times as you want!


	<script>
		for (let i = 0; i < 10; i++) {
			squidFact();
		} // outputs "Squid are ancient animals." 10 times.
	</script>

Arguments

Functions can accept input values, called arguments.


	<script>

Returning Values

Functions return 'undefined' by default. You need to explicitly return values.


	<script>
	function square(num) {
		return num * num;
	}

	function badSquare(num) {
		num * num
	}

	square(4); // 16
	square(5); // 25

	badSquare(4); // undefined
	badSquare(5); // undefined

	</script>

Let's Develop It

Add a return statement to your name function. Use that function to set the value of a variable.
